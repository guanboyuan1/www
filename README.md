# 贪吃蛇智能体 🐍

一个基于 A* 寻路算法的贪吃蛇 AI 智能体项目。

## 项目简介

这是一个使用 HTML5 Canvas 和 JavaScript 开发的贪吃蛇游戏，内置 AI 智能体可以自动玩游戏。

## 功能特性

- 🎮 **经典贪吃蛇游戏**：完整的游戏逻辑和流畅的动画效果
- 🤖 **AI 智能体**：使用 A* 寻路算法自动寻找食物
- ⚡ **模式切换**：支持手动模式和 AI 模式之间切换
- 📊 **分数系统**：实时得分和最高分记录（本地存储）
- 🎨 **精美界面**：深色主题设计，视觉效果出色

## 技术实现

### AI 算法
- **A* 寻路算法**：通过启发式搜索找到最短路径
- **曼哈顿距离**：作为启发函数评估路径代价
- **安全检测**：当无法找到路径时，自动寻找安全位置移动

### 游戏特性
- 实时得分统计
- 最高分本地存储
- 暂停/继续功能
- 游戏状态提示

## 使用方法

### 在线预览
直接打开 `index.html` 文件即可运行游戏。

### 操作说明
| 按钮 | 功能 |
|------|------|
| 开始游戏 | 开始游戏（手动模式） |
| 暂停 | 暂停/继续游戏 |
| 重置 | 重新开始游戏 |
| 切换AI模式 | 开启/关闭 AI 自动玩游戏 |

### 键盘控制
- **方向键** 或 **WASD**：控制蛇的移动方向

## 开发

### 本地运行
```bash
python -m http.server 8000
```
然后访问 http://localhost:8000

### 游戏速度
游戏速度可在 `snake.js` 的 `start()` 方法中调整 `setInterval` 的时间间隔。
